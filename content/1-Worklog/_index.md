This is a worklog that records in detail my entire internship process at Amazon Web Services Vietnam Company Limited. The internship program and project deployment were completed within **11 weeks**, closely following the team's overall schedule. This process includes stages ranging from getting familiar with the environment and developing the "DaiMarket" project (a cloud-based marketplace for digital products and 3D models) to the period of collaborating and assisting the team leader in deploying the shared system onto the actual Cloud environment.

Below is an overview of the tasks completed each week:

**Week 1:** [Project onboarding, React/Vite initialization, and PostgreSQL setup](1.1-week1/)

**Week 2:** [UI/UX wireframing, component planning, and Database schema design](1.2-week2/)

**Week 3:** [Static layouts, Prisma Schema integration, and Database relationships](1.3-week3/)

**Week 4:** [API preparation, State Management, and SQL seed data](1.4-week4/)

**Week 5:** [Product modules, UI development, and SQL query optimization](1.5-week5/)

**Week 6:** [Frontend refactoring, advanced Data Models, and constraints](1.6-week6/)

**Week 7:** [3D Viewer integration and S3 metadata database updates](1.7-week7/)

**Week 8:** [Checkout UI, SePay integration, and transaction logging](1.8-week8/)

**Week 9:** [E2E Prototype testing, Responsive UI, and Soft Delete logic](1.9-week9/)

**Week 10:** [Library flow, JWT authorization, and Amazon RDS preparation](1.10-week10/)

**Week 11:** [Vercel deployment, API rewrites, and RDS data verification](1.11-week11/)
