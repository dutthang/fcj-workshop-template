This is a worklog that records in detail my entire internship process at Amazon Web Services Vietnam Company Limited. The internship program and project deployment were completed within **12 weeks**, closely following the team's overall schedule. This process includes stages ranging from getting familiar with the environment and developing the "Smart Food" project (A fast-food ordering application using AI for recommendations) to the period of collaborating and assisting the team leader in deploying the shared system onto the actual Cloud environment.

Below is an overview of the tasks completed each week:

**Week 1:** [Get acquainted with the internship, teamwork, and explore the basic AWS platform](1.1-week1/)

**Week 2:** [Explore pricing mechanisms, study Module 01, and basic IAM](1.2-week2/)

**Week 3:** [Practice EC2, S3, SSH services and test deploy a sample application](1.3-week3/)

**Week 4:** [Analyze requirements, design system architecture, and initialize the Smart Food project](1.4-week4/)

**Week 5:** [Set up the environment on Visual Studio Code, code API skeleton, initial UI, and research S3 storage](1.5-week5/)

**Week 6:** [Stabilize the Smart Food project structure, finalize the data model, and AI suggestion flow](1.6-week6/)

**Week 7:** [Explore Serverless architecture (AWS Lambda, DynamoDB) and handle Real-time connections](1.7-week7/)

**Week 8:** [Finalize features, optimize performance, and configure HTTPS security](1.8-week8/)

**Week 9:** [Load testing (Stress Testing), evaluate prototype, and prepare the AWS deployment environment](1.9-week9/)

**Week 10:** [Deploy Smart Food to EC2 with Nginx Web Server. Start assisting the team leader with DaiMarket system testing](1.10-week10/)

**Week 11:** [Collaborate with the team leader on the practical integration of the DaiMarket system (Vercel frontend, EC2 API, RDS, S3) and troubleshoot Production errors](1.11-week11/)
