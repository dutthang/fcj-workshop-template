Đây là worklog ghi chép chi tiết toàn bộ quá trình thực tập của mình tại Công ty TNHH Amazon Web Services Việt Nam. Chương trình thực tập và triển khai dự án được hoàn thành trong vòng **11 tuần**, bám sát theo tiến độ chung của team. Quá trình này bao gồm các giai đoạn từ làm quen môi trường, phát triển dự án "DaiMarket" (nền tảng marketplace trên cloud cho sản phẩm số và mô hình 3D) cho đến giai đoạn phối hợp và hỗ trợ team leader triển khai hệ thống chung lên môi trường Cloud thực tế.

Dưới đây là tổng quan các công việc đã hoàn thành qua từng tuần:

**Tuần 1:** [Làm quen dự án, khởi tạo React/Vite và thiết lập PostgreSQL](1.1-week1/)

**Tuần 2:** [Phác thảo UI/UX, lên kế hoạch component và thiết kế Database schema](1.2-week2/)

**Tuần 3:** [Dựng layout tĩnh, tích hợp Prisma Schema và thiết lập quan hệ CSDL](1.3-week3/)

**Tuần 4:** [Chuẩn bị API, State Management và tạo dữ liệu mẫu SQL (seed data)](1.4-week4/)

**Tuần 5:** [Module sản phẩm, phát triển giao diện và tối ưu truy vấn SQL](1.5-week5/)

**Tuần 6:** [Refactor Frontend, hoàn thiện Data Model nâng cao và các ràng buộc](1.6-week6/)

**Tuần 7:** [Tích hợp 3D Viewer và cập nhật database lưu metadata của S3](1.7-week7/)

**Tuần 8:** [Giao diện thanh toán (Checkout), tích hợp SePay và ghi log giao dịch](1.8-week8/)

**Tuần 9:** [Kiểm thử E2E Prototype, xử lý Responsive UI và logic Soft Delete](1.9-week9/)

**Tuần 10:** [Luồng thư viện cá nhân, xác thực JWT và chuẩn bị dữ liệu cho Amazon RDS](1.10-week10/)

**Tuần 11:** [Deploy lên Vercel, cấu hình API rewrites và kiểm chứng dữ liệu trên RDS](1.11-week11/)
